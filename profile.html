Description

This is a PROF-formalised version of DCAT-AP 2.0.1, the DCAT Application Profilefor data portals in Europe, version 2.0.1. This is done so DCAT-AP resources - validators, images and so on - can be associated with PROF constructs which are necissary for some tooling.
